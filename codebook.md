# Data Set Information
The data set presented with in this work, it is the summary of the data collected by weareable products (from the accelerometers from the Samsung Galaxy S smartphone). For more information about how these data were captured, you can refer to the original website [UCI-ML Repository](http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones). 

In case you want to You can download the original data set, you can use the following link for it: [ORIGINAL-WEAREABLE-DATA](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip) 

## DATA Treatment
The original data, was treated as follow:

1. The training and the test sets were merged in one data set.

2. Only the measurements on the mean and standard deviation for each measurement were extracted.

3. The Activity values were replaced by their description.

4. Appropiate descriptive labels were added to each variable name.

5. Then, an  average of each variable for each activity and each subject was prepared. 
*RESULT:*  The data set summarizing the average value for each variable for each activity and each subject.
